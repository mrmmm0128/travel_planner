# 🧭 トラベルプランナーマッチングアプリ

## 📌 概要
旅行者と「自分に合った旅行プランナー」をマッチングし、  
チャットを通じて旅行プランを共同作成できるモバイルアプリ。

---

## 👤 ユーザー

- 旅行者
- プランナー
- ガイド（※プランナーと統合するか検討中）

---

## ⚙️ 主な機能

### 🔐 ユーザー関連
- ユーザー登録 / ログイン

### 🔎 プランナー選択
- エリア
- 得意ジャンル（例：グルメ、観光、アクティビティ）
- 予算感
- メンバー構成（例：子連れ、カップル、一人旅）

### 💬 プラン作成
- チャットでプランナーとやり取り
- 旅行プランの共同作成

### 📖 プラン確定
- しおり生成
- スケジュール管理

### 🧳 旅行当日
- しおりを見ながら旅行進行

### 🤖 （検討中）
- プランナー依頼前の簡易プラン自動生成
